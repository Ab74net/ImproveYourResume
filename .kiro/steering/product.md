# Product Overview

Resume Reviewer Studio is a web application that helps job candidates improve their resumes for specific role and company targets.

## Core Value Proposition

Upload a resume, target a role and company, paste a job description, and receive structured feedback including:
- ATS-style match score and keyword coverage
- Missing skills and weak signals
- Bullet point rewrites with action verbs and metrics
- Company-specific advice tailored to Amazon, Google, Meta, Apple, Netflix, or general FAANG style
- Practical project suggestions to close skill gaps

## Target Users

- Job seekers targeting FAANG and tech companies
- Students and new grads preparing for internships or entry-level roles
- Professionals pivoting between roles (SWE, QA, Data Analyst, Product Manager, DevOps, Security)

## Current Stage

Phase 1 prototype with frontend-only implementation. AI calls happen in the browser for demo speed. Phase 2 will move analysis to AWS backend (API Gateway + Lambda).

## Key Features

- Role-based targeting (SWE, QA, Data Analyst, Product Manager, DevOps, Security)
- Company-specific rubrics (Amazon, Google, Meta, Netflix, Apple, General FAANG)
- PDF upload with text extraction
- Sample job descriptions for quick testing
- Session-based analysis history (current browser session only)
- Demo mode with before/after examples
